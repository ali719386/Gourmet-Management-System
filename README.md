# Gourmet Management System

The **Gourmet Management System** is a full-stack web application designed to efficiently manage the operations of a gourmet food business — from product inventory and categories, to orders, customers, and reporting. This system provides an intuitive web interface for administrators and staff to oversee key workflows, ensuring high quality, streamlined processes, and data-driven decisions.

---

## 🚀 Features

- User authentication & role-based access (admin, staff)  
- Manage gourmet products: add/edit/delete product entries with descriptions, pricing, categories and images  
- Inventory tracking & update notifications  
- Customer order management: view orders, change status (pending → fulfilled → delivered)  
- Dashboard with key metrics and reports (sales, popular items, inventory status)  
- Responsive UI allowing use on desktop and mobile devices  
- Built with scalability in mind: easy to extend with features like payment gateway, analytics, and multi-store support  

---

## 🧰 Technology Stack

- **Backend:** Django (Python)  
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap  
- **Database:** SQLite for development (can be switched to PostgreSQL or MySQL for production)  
- **Version Control:** Git & GitHub  
- Optionally deployable to cloud services like Heroku, Render, DigitalOcean.
